# Deep Packet Inspection System

This project implements a Deep Packet Inspection (DPI) system to analyze network traffic using packet capture data.

## Features
- Captures and analyzes network packets
- Works with .pcap files
- Inspects packet headers and payload
- Helps in understanding network traffic patterns

## My Work
- Implemented and executed the packet analyzer
- Studied packet structure and TCP/IP protocols
- Explored how packet parsing and inspection works
- Modified code for better understanding and testing

## Tech Stack
- C++
- Networking Concepts (TCP/IP)
- Packet Analysis

## How to Run
1. Compile the code using a C++ compiler
2. Provide a .pcap file as input
3. Run the executable to analyze packets
